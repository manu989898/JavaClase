Caja registradora v.2

Sobre este proyecto

Primer proyecto de java. Es una aplicación para un sistema de caja registradora para restaurantes.

Funcionalidades

  • Gestión de reservas : Permite añadir, eliminar y modificar reservas segun el nº de mesa, la cantidad de personas, el titular y la hora de llegada.
  • Impresión de ticket : Permite guardar la información de articulos consumidos y contabilidad en un archivo de documento tipo .txt.
  • Gestión de reservas: Permite añadir, eliminar y modificar reservas segun el nº de mesa, la cantidad de personas, el titular de la reserva y la hora de llegada.
